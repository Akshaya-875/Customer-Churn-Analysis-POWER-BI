# Customer Churn Analysis – Power BI

## Project Overview
This project analyzes customer churn in a telecom company using Power BI. 
The objective is to identify churn patterns, key drivers, and high-risk customer segments to support proactive retention strategies.

## Tools Used
- Power BI
- DAX
- CSV Dataset

## Key Features
- Customer tenure distribution analysis
- Churn analysis by contract type and gender
- Monthly charges vs churn comparison
- Churn Rate KPI using DAX
- Identification of high-risk customer segments
- Interactive slicers for dynamic analysis

## Files in Repository
- Customer_Churn_Analysis.pbix – Power BI dashboard
- telecom_customer_churn.csv – Dataset
- /screenshots – Dashboard screenshots
- Project report (PDF)

## Outcome
The analysis shows that customers with low tenure, month-to-month contracts, and higher monthly charges are more likely to churn.
